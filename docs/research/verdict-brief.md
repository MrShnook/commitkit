**Date:** 2026-03-01
**Verdict:** 🟢 **ADVANCE TO PROTOTYPING**

## The Opportunity

**220,000 US families** are actively navigating college soccer recruitment at any given time. They spend **$5K-20K/year** on the process using spreadsheets, email, and overpriced services like NCSA ($1K-4K/yr). No tool exists that helps families **manage the process** — every competitor is a coach-focused marketplace or profile platform.

## Why Now

1. **NCSA's reputation is cratering** — BBB complaints, parent backlash, coach distrust. Families want an alternative.
2. **House v. NCAA settlement** (July 2025) rewrote the rules — 28-player roster caps, new scholarship structures create more complexity.
3. **Youth soccer is booming** — 850K+ HS players, 3% YoY growth. MLS Next, ECNL, GA leagues expanding.
4. **No family-side CRM exists.** ProductiveRecruit (closest) is B2B (clubs/orgs), not B2C (individual families).

## The Product

**CommitKit** = Recruitment CRM for soccer families.
- **School Kanban:** Dream → Target → Safety → Contacted → Responded → Visited → Offered → Committed
- **Coach Contacts:** Track outreach, responses, follow-ups across every program
- **Eligibility Timeline:** NCAA/NAIA core courses, test scores, registration deadlines
- **Showcase Calendar:** Camps, ID clinics, college visits
- **Email Templates:** Coach outreach, follow-up, thank you

## Target Customer

Parents of competitive youth soccer players (MLS Next, ECNL, GA, varsity) ages 14-18. Currently using spreadsheets. Spending $10K+/year on recruitment. Willing to pay $19/mo for a tool that reduces chaos.
