# CommitKit ⚽

Soccer recruitment management platform for families and teams.

> Track outreach, manage school lists, stay on top of eligibility timelines.

**Stage:** Researching

## What Is This?

College soccer recruitment is a fragmented, high-stress process. Existing tools (NCSA, SportsRecruits, FieldLevel) are expensive, coach-focused, and don't solve the family's workflow problem.

CommitKit is a **family-side recruitment CRM** — the command center for managing the entire recruiting journey.

## Features (Planned)

- 📋 School tracking board (Kanban pipeline)
- 📧 Coach contact management
- 📅 Eligibility timeline tracking
- 🎥 Highlight reel manager
- 🏕 Showcase & camp calendar
